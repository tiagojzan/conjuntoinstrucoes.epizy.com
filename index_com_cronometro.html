Vídeo eChalk.co.uk

Quer saber como um computador executa um programa?

Se você realmente quer entender o que acontece quando um programa é executado, este é o lugar ideal para começar. Esta simulação fantástica fornece um modelo conceitual simples do funcionamento interno de um computador.
Seu desafio é construir um programa curto para resolver um problema simples e, ao fazer isso, você descobrirá a relação entre os registradores, RAM, barramento do sistema e conjunto de instruções.
Versão apenas para demostração.
